# 테스트용 로그인 계정

개발/테스트 시 사용할 수 있는 이메일 로그인 계정입니다.

## 계정 정보

| 항목 | 값 |
|------|-----|
| **이메일** | `test@test.com` |
| **비밀번호** | `Test1234!` |

## 사용 방법

### 1. 테스트 계정 생성 (최초 1회)

**방법 A – 브라우저로 실행 (가장 쉬움)**  
배포된 사이트나 로컬 서버에서 아래 주소로 **브라우저 주소창에 입력**해 접속합니다.

- **배포 서버**: `https://ailand.dothome.co.kr/api/auth/seed-test-user`
- **로컬**: `http://localhost/api/auth/seed-test-user` (또는 사용 중인 도메인)

한 번만 열면 테스트 계정이 생성되고, 화면에 이메일/비밀번호가 JSON으로 표시됩니다. (완료 후 보안을 위해 `config/routes.php`에서 해당 라우트를 제거해도 됩니다.)

*참고: `/seed_test_user.php` 경로는 서버 설정에 따라 404가 날 수 있어, 위 API 주소를 사용하는 것을 권장합니다.*

---

**방법 B – 터미널(명령 프롬프트)에서 PHP 실행**  
PC에 PHP가 설치되어 있다면, 프로젝트 루트 폴더에서 아래처럼 실행합니다.

1. **명령 프롬프트** 또는 **PowerShell**을 연다.
2. 프로젝트 폴더로 이동한다.  
   예: `cd C:\Users\IBK\OneDrive\바탕 화면\News`
3. 다음 명령을 입력한다.

   ```bash
   php database/seed_test_user.php
   ```

- PHP가 설치되어 있지 않으면 “php는(은) 내부 또는 외부 명령으로 인식되지 않습니다” 같은 메시지가 나옵니다. 이 경우 **방법 A(브라우저)** 를 사용하면 됩니다.
- `users` 테이블에 `password_hash` 컬럼이 없으면 자동으로 추가됩니다.
- 이미 `test@test.com` 계정이 있으면 메시지만 출력하고 건너뜁니다.

### 2. 로그인  
   - 로그인 페이지(`/login`)에서 **이메일**과 **비밀번호**를 입력한 뒤 로그인 버튼을 누릅니다.
   - 카카오 로그인 없이 위 계정으로 로그인할 수 있습니다.

## 주의사항

- **운영 환경에서는 해당 계정을 삭제하거나 비밀번호를 변경**하는 것을 권장합니다.
- 테스트 계정은 즐겨찾기(북마크), My Page 등 로그인 필요 기능을 확인할 때 사용하세요.
