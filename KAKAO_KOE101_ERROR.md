# 카카오 앱 관리자 설정 오류 (KOE101) 해결 가이드

## ❌ 오류: 앱 관리자 설정 오류 (KOE101)

이 오류는 카카오 개발자 콘솔에서 앱 관리자 정보가 설정되지 않았거나, 앱 상태가 비정상일 때 발생합니다.

## ✅ 해결 방법 (단계별)

### 1단계: 앱 관리자 정보 설정

1. [Kakao Developers](https://developers.kakao.com) 접속
2. **내 애플리케이션** 선택
3. **앱 설정** → **앱 관리자** 메뉴로 이동
4. 다음 정보를 모두 입력:
   - **앱 관리자 이름** (필수)
   - **앱 관리자 이메일** (필수)
   - **앱 관리자 전화번호** (필수)
   - **앱 관리자 소속** (선택)
5. **저장** 클릭

### 2단계: 앱 기본 정보 확인

1. **앱 설정** → **앱 기본정보** 메뉴로 이동
2. 다음 항목이 모두 입력되어 있는지 확인:
   - **앱 이름** (필수)
   - **앱 아이콘** (필수)
   - **앱 설명** (필수)
   - **앱 카테고리** (필수)
3. 누락된 항목이 있으면 모두 입력 후 **저장**

### 3단계: 앱 상태 확인

1. **앱 설정** → **앱 상태** 메뉴로 이동
2. 앱 상태가 다음 중 하나인지 확인:
   - ✅ **개발** (테스트용)
   - ✅ **운영** (서비스용)
3. 앱 상태가 **준비중** 또는 **비활성화**인 경우:
   - **개발** 또는 **운영**으로 변경
   - 변경 사유 입력
   - **저장** 클릭

### 4단계: 플랫폼 등록 확인

1. **앱 설정** → **플랫폼** 메뉴로 이동
2. **Web 플랫폼**이 등록되어 있는지 확인
3. 등록되지 않았다면:
   - **Web 플랫폼 등록** 클릭
   - 사이트 도메인 입력: `http://ailand.dothome.co.kr`
   - **저장** 클릭

### 5단계: 카카오 로그인 제품 활성화

1. **제품 설정** → **카카오 로그인** 메뉴로 이동
2. **활성화 설정** → **ON**으로 변경
3. **저장** 클릭

## 🔍 체크리스트

다음 항목을 모두 확인하세요:

- [ ] **앱 관리자 정보**: 앱 설정 → 앱 관리자 → 모든 필수 정보 입력됨
- [ ] **앱 기본 정보**: 앱 설정 → 앱 기본정보 → 모든 필수 정보 입력됨
- [ ] **앱 상태**: 앱 설정 → 앱 상태 → "개발" 또는 "운영" 상태
- [ ] **플랫폼 등록**: 앱 설정 → 플랫폼 → Web 플랫폼 등록됨
- [ ] **카카오 로그인 활성화**: 제품 설정 → 카카오 로그인 → 활성화 설정 ON

## ⚠️ 주의사항

1. **앱 관리자 정보는 필수입니다**
   - 이름, 이메일, 전화번호는 반드시 입력해야 합니다
   - 실제 사용 가능한 정보를 입력하세요

2. **앱 상태 변경 시 시간이 걸릴 수 있습니다**
   - "준비중" → "개발" 또는 "운영"으로 변경 시
   - 카카오 측 검토가 필요할 수 있습니다 (보통 즉시 반영)

3. **앱 기본 정보는 모두 입력해야 합니다**
   - 앱 이름, 아이콘, 설명, 카테고리는 필수 항목입니다

## 🧪 설정 완료 후 확인

### 1. 카카오 개발자 콘솔에서 확인

1. **앱 설정** → **앱 상태** 메뉴
2. 오류 메시지가 사라졌는지 확인
3. 앱 상태가 정상적으로 표시되는지 확인

### 2. 테스트 페이지에서 확인

```
http://ailand.dothome.co.kr/api/auth/test.php
```

이 페이지에서:
- REST API 키가 정상적으로 표시되는지 확인
- "카카오 로그인 테스트" 버튼이 활성화되어 있는지 확인

### 3. 직접 로그인 테스트

1. 프론트엔드에서 "카카오로 시작하기" 버튼 클릭
2. 카카오 로그인 페이지가 정상적으로 열리는지 확인
3. 로그인 후 콜백이 정상적으로 처리되는지 확인

## 📞 추가 도움

문제가 계속되면:

1. **카카오 개발자 센터 문의**
   - [카카오 개발자 센터](https://developers.kakao.com/support)
   - 오류 코드: KOE101

2. **앱 삭제 후 재생성** (최후의 수단)
   - 기존 앱을 삭제하고 새로 생성
   - 모든 설정을 처음부터 다시 진행

## 🔄 일반적인 해결 순서

1. ✅ 앱 관리자 정보 입력 (가장 중요!)
2. ✅ 앱 기본 정보 입력
3. ✅ 앱 상태를 "개발" 또는 "운영"으로 변경
4. ✅ Web 플랫폼 등록
5. ✅ 카카오 로그인 제품 활성화
6. ✅ Redirect URI 등록
7. ✅ 동의항목 설정

이 순서대로 진행하면 KOE101 오류가 해결됩니다.
